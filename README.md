### Шаг 4
Разметили LLM моделью Giga Chat 3.1 датасет, в котором оставили только ~2000 российских макроэкономических новостей уже на определение динамики макроэкономических факторов из новостей. Для этого использовали промпт PROMPT_MACRO_FACTORS_DINAMICS.

**Выделяли следующие показатели:**

Ключевые показатели экономики:
- процентная ставка
- ВВП
- инфляция
- безработица

Производство:
- капитал
- инвестиции
- производство

Население:
- Потребление
- численность рабочей силы
- сбережения
- заработные платы
- доходы населения

Внешне экономическая политика:
- валютный 
- импорт
- экспорт

Государство:
- государственные расходы
- государственный долг
- дефицит бюджета


Итого для 18 показателей выделили:
- up - показатель вырос
- down - показатель сократился
- not stated - показатель отсутствует в новости или имеет неопределенную динамику

### Шаг 6
Перевели в код макроэкономические модели, которые будут работать с заданными параметрами
- IS-LM-BP
- AD-AS
- IS-MP-PC
- Солоу с человеческим капиталом
- Модель Рамсея

Выводы динамики макро показателей подставляются в промпт для агрегации результатов. Это было сделано, чтобы избежать ошибок LLM при попытках подсчета параметров в моделях.
